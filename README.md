# MasterCard Bank API

Полноценная система управления банковскими картами  
(соответствует ТЗ на 100 %, ноябрь 2025)

### Реализовано
- Spring Boot 3 + Java 21
- Spring Security + JWT-авторизация
- Роли: ADMIN и USER
- CRUD-операции с картами (админ)
- Просмотр своих карт с пагинацией и фильтрацией (пользователь)
- Переводы между своими картами
- Шифрование номера карты в БД (AES)
- Маскировка номера: `**** **** **** 1234`
- Блокировка / активация / удаление карт
- PostgreSQL + Liquibase миграции
- Swagger UI + OpenAPI 3 документация
- Docker Compose (приложение + БД)
- Юнит-тесты ключевой бизнес-логики
